# Until Dawn (UDawn) Hook
A menu tool for Until Dawn Remake.

## Features

| Feature                       | Description |
|-------------------------------|------------------------------------------------------------|
| Character swapper             | Swap a character with the specified character and variant. |
| Swap rule maker               | Create a custom rule which swaps a specific character's variant for another. |
| Custom menu character         | Replace the main menu character with a custom one. |
| Free camera                   | Move the camera around the world freely and explore. |
| First-person camera           | Replace the third-person camera with a first-person view. |
| Facial animation strength     | Control the strength of facial animations. |
| Custom game speed/freeze time | Control game speed/freeze game time. |
| Enable sprinting              | Adds the ability to sprint in-game. |
| Enable debug menu             | Enable the game's hidden debug menu. |

## Installation

**UDawnHook supports only the latest Steam version of the game. Support won't be provided if you are on an older version of the game.**

Drag and drop extracted files into the **Until Dawn\Windows\Bates\Binaries\Win64** folder.

You can find your game's directory in Steam by right-clicking the game->Manage->Browse local files.

## Usage
Open the menu by pressing the **Insert** key while in-game.
This can be configured in **UDawnhook_user.ini** or from the menu's **Settings** tab.
